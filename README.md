# 🏢 대건명가 웹사이트

대건명가 브랜드의 정보를 전달하고, 잠재 고객의 문의를 효과적으로 수집하기 위해 제작된 **반응형 웹사이트**

---

## 🚀 주요 특징

- **반응형 UI (Responsive Design):** HTML, CSS, JavaScript를 사용하여 PC, 태블릿, 모바일 등 모든 기기에서 최적화된 화면을 제공합니다.
- **상시 노출 '문의하기' 버튼:** 기기에 상관없이 '문의하기' 버튼을 화면에 고정 배치하여 사용자의 행동 유도(CTA)를 극대화했습니다.
- **다양한 연락 방법 제공:** 버튼 클릭 시 인스타그램, 이메일, 전화 등 다양한 방법을 먼저 제시하여 사용자 편의성을 높였습니다.
- **보안 강화 문의 폼:** 스팸 방지를 위한 다각도 보안 로직이 적용되었습니다.

## 🛠 기술 스택

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Google Apps Script, Google Sheets API

## 🛡 스팸 방지 및 보안 전략

단순한 데이터 전송을 넘어, 악의적인 스팸 및 봇의 공격을 막기 위해 다음 두 가지 전략을 사용합니다.

1. **허니팟(Honeypot) 전략:** - 사용자에게는 보이지 않지만 봇은 인식할 수 있는 숨겨진 입력 필드(`display: none;`)를 배치합니다. 
   - 이 필드에 값이 채워져 있다면 봇으로 간주하고 제출을 원천 차단합니다.
   
2. **세션 기반 제출 제한:**
   - 무분별한 도배를 막기 위해 한 세션당 문의 제출 횟수를 **최대 3회**로 제한합니다.
   - submitCount 변수를 활용해 세션별 제출 횟수를 카운트하며, 페이지 새로고침 시 초기화됩니다.

## 📂 프로젝트 구조

```text
├── Assets/
│   ├── favicon.jpg      # 웹사이트 파비콘
│   ├── logo.jpg         # 대건명가 로고
│   └── 프로토타입.txt    # 기획 데이터
├── css/
│   ├── contact.css      # 문의하기 페이지 스타일
│   └── index.css        # 메인 페이지 스타일
├── js/
│   ├── contact.js       # 문의 폼 로직 및 보안 설정
│   └── script.js        # 메인 인터랙션 스크립트
├── .gitattributes
├── contact.html         # 문의하기 페이지
└── index.html           # 메인 랜딩 페이지

사용 방법:

1. index.html을 브라우저에서 실행하여 웹사이트를 확인합니다.

2. 화면에 고정된 '문의하기' 버튼을 누르면 채널 선택 섹션이 나타납니다.

3. **'지금 바로 문의하기'**를 클릭하면 contact.html로 이동하여 폼을 작성할 수 있습니다.

4. 작성된 내용은 실시간으로 관리자의 Google Spreadsheet에 기록됩니다.
