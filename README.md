# 🏢 대건명가 B2B 웹사이트

대건명가 브랜드의 정보를 효과적으로 전달하고, 잠재 고객의 문의를 체계적으로 수집하기 위해 제작된 **반응형 웹사이트 프로젝트**입니다.

---

## ⭐ 주요 특징

* **반응형 UI (Responsive Design):** HTML, CSS, JavaScript를 활용하여 PC, 태블릿, 스마트폰 등 다양한 기기 환경에 최적화된 화면을 제공합니다.
* **제품 목록:** 판매 중인 제품들을 확인할 수 있습니다.
* **상시 노출 '문의하기' 버튼:** 기기 종류와 상관없이 화면에 문의 버튼을 상단에 고정 배치하여 사용자의 행동 유도(CTA)를 극대화했습니다.
* **다양한 연락 수단 제공:** 버튼 클릭 시 인스타그램, 이메일, 전화 등 다양한 연락 방법을 먼저 제시하여 유저 편의성을 높였습니다.
* **실시간 데이터 연동 및 알림:** 문의 폼(form) 제출 시 정보가 **Google Spreadsheet**에 즉시 기록되며, 동시에 관리자에게 **이메일 알림**이 전송됩니다.

## 🛠 기술 스택

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Google Apps Script, Google Sheets API

## 🛡 스팸 방지 및 보안 전략

악의적인 스팸 및 봇 공격을 차단하기 위해 이중 보안 로직을 적용했습니다.

1. **허니팟(Honeypot) 전략:**
   - 사용자에게는 보이지 않지만 봇은 인식할 수 있는 숨겨진 입력 필드(`display: none;`)를 배치합니다.
      - 해당 필드에 값이 입력된 채 제출되면 봇으로 간주하여 전송을 차단합니다.

2. **세션 기반 제출 제한:**
   - 무분별한 도배를 방지하기 위해 한 세션당 문의 제출 횟수를 **최대 3회**로 제한합니다.
   - `submitCount` 변수를 활용해 세션별 제출 횟수를 카운트하며, 페이지 새로고침 시 초기화됩니다.

## 📂 프로젝트 구조

```text
├── Assets/
│   ├── favicon.jpg      # 웹사이트 파비콘
│   ├── logo.jpg         # 대건명가 로고
│   └── 프로토타입.txt     # 기획 예시
├── css/
│   ├── contact.css      # 문의하기 페이지 스타일
│   └── index.css        # 메인 페이지 스타일
├── js/
│   ├── contact.js       # 문의 폼 로직 및 보안 설정
│   └── script.js        # 메인 인터랙션 스크립트
├── .gitattributes
├── contact.html         # 문의하기 페이지
└── index.html           # 메인 랜딩 페이지
```

## 💡 사용 방법

1. Github에서 zip 파일을 **다운로드**합니다.
2. 다운로드한 **파일을 압축 해제**합니다.
3. `index.html` 파일을 **더블 클릭**하여 브라우저에서 웹사이트를 실행합니다.
4. 화면에 고정된 **'문의하기'** 버튼을 누르면 다양하게 연락할 수 있는 섹션이 나타납니다.
5. **지금 바로 문의하기**를 클릭하여 `contact.html`로 이동한 후 폼을 작성하고 제출합니다.
